AI / 머신러닝 엔진

ML은 "결정"이 아니라 "확률 필터"

의사결정 구조
  • ① ML 모델이 상승 확률 예측
  • ② Rule 엔진이 조건으로 필터링
  • ③ 두 단계를 통과해야 최종 매수/매도
  • 입력 피처 · OHLCV, MA5/20, 변동성, 수익률 변화율
# 예시 로직 if model.predict(stock_data) > 0.7: if volume_spike == True: BUY
ML이 적용되는 3가지 포인트
  • 매수 확률 예측 — "지금 오를 확률" 계산
  • 종목 필터링 — 좋은 종목 vs 나쁜 종목 구분
  • 리스크 예측 — 급락 가능성 탐지
  • 분석 지표 — 수익률, MDD, CAGR
AI 챗봇 기반 전략 설정

말로 하면, 전략 파라미터로 변환된다

사용자: "RSI 65 이상일 때 매수해줘" │ GPT 해석 엔진 │ { "indicator": "RSI", "threshold": 65, "condition": "above", "action": "BUY" } │ AI 트레이딩 Agent에 전략 자동 적용
코드나 설정 파일을 직접 수정하지 않아도, "내 말 그대로" 전략이 만들어지는 대화형 인터페이스.
